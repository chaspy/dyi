# dyi

"Discuss Your Interest" with Rails 6.0 Application

This app aims to share the content of the discussion in the event for discussion.

## Dependencies

- Ruby 2.6.2
- Ruby on Rails 6.0.0
- Postgresql 9.5

## Getting stated

### on local

```
$ bundle
$ rails s
```
